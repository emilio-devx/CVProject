# TO-DO App

Aplicación de tareas creada para practicar fundamentos de React como estado, renderizado condicional, filtrado y persistencia con localStorage.

## Funcionalidades
- Añadir tareas
- Editar tareas
- Eliminar tareas
- Marcar como completadas
- Filtrar por estado
- Persistencia de datos con localStorage

## Tecnologías
- React
- Tailwind CSS
- Vite

## Lo que practico con este proyecto
- gestion de estado con `useState`
- renderizado de listas con `.map()`
- edición y actualización de elementos
- filtrado dinámico de datos
- persistencia de datos con `localStorage`
- composición de interfaz con Tailwind CSS

## Instalación

### 1. Instalar dependencias del frontend

```bash
npm install
```

### 2. Iniciar el frontend

```bash
npm run dev
```
